如果您拥有一个出租房产或正在出租的房产，您可以通过申请各种投资房产税收减免来改善您的财务状况。您可能已经在申请一些减免，但您是否利用了所有可用的税收减免渠道来抵扣租金收入？

确保您获得所有可能的福利，可能是您在投资房产中获利与亏损之间的关键。因此，我们将为您详细解析在将投资房产作为生产性资产时可以申请的所有税收减免。

### 贷款利息

作为借款人，您可以将贷款的利息还款作为投资房产的税收减免。此外，如果您的投资房产贷款涉及任何借款费用，例如银行费用和手续费，这些费用也可以在税前扣除。

### 折旧 （Depreciation）

房产的正常磨损是不可避免的，并且会自然降低其财务价值。折旧是一项可抵扣的房产税收减免，用以弥补这一损失，您可以将其抵扣在投资房产的租金收入中。折旧包括以下两种类型：

### 建筑工程折旧（Capital Works Depreciation）

任何在1985年7月17日之后建造的投资房产，均有资格获得建筑折旧费用的税收减免。

装修中的建筑成本也可以作为投资房产的税收减免。但这些费用并不能像其他费用那样完全抵扣，而是按比例分几年申报，例如每年2.5%，持续40年。

### 设备和装置折旧（Plant and Equipment Depreciation）

您还可以对出租房产内外的固定装置和设备进行折旧抵扣，这些设备随着使用而磨损，如橱柜、地毯、烤箱、空调和淋浴设施。

### 工程造价师费用（Quantity Surveyor）

为了您的最佳利益，建议您寻求专业的工程造价师建议。他们将为您的房产制定税收折旧计划，以记录价值损失，同时您还可以在申请折旧减免时包括他们收取的费用。

## 租赁物业费用

作为房东，您需要承担与投资房产相关的许多费用。许多这些费用可以在支付当年的税收中申报为租赁物业减免，例如维护和园艺费用以及物业管理费。

| 费用类别 | 描述 |
|---------|------|
| **广告费用** | 在空置期间，房东需要支付广告费用以寻找租户。如果您在广告上花费了金钱，您可以申请这笔费用的税收减免。 |
| **法律费用** | 法律咨询是常见的投资房产减免之一。您可能需要律师来准备租赁文件或帮助您获得驱逐令。 |
| **租赁代理费用** | 如果您雇佣了物业代理来管理您的房产并维护与租户的良好关系，您可以将此费用申报为税收减免。 |
| **税务咨询** | 使用税务顾问确保您的税款正确提交，产生的费用可以立即申报为税收减免。 |
| **水电煤费用 (Utilities)** | 如果您支付了水电气等费用，而非由租户承担，这些费用可以申报为税收减免。 |
| **土地税** | 当您出租房产时，土地税可以作为税收减免申报。 |
| **维修和维护** | 用于维护房产的费用（而非改善房产）可以申报为投资房产的税收减免。 |
| **虫害控制** | 如果您聘请了虫害控制人员来清除影响房产健康和安全的害虫，相关费用可以立即申报为税收减免。 |
| **物业保险** | 租赁保险可以保护您的租赁房产及其内容物，这是一个可税前扣除的费用。 |
| **市政费 (Council Rates)** | 如果市政费由您而非租户支付，您可以将其申报为投资房产的税收减免。 |
| **清洁费用** | 无论是每周支付的清洁服务费用，还是租户搬出后进行的清洁费用，您都可以将其申报为投资房产的税收减免。 |
| **物业管理费 (Body Corporate Fees)** | 如果您的投资房产是单元或联排别墅，且物业管理费由您而非租户支付，您可以将其申报为投资房产的税收减免。 |
| **园艺费用** | 如果用于维护、园艺和更换植物或花园结构的费用没有改善花园，这些费用是可以申报的。 |
| **文具、电话和互联网费用** | 与管理您的投资房产直接相关的任何文具、电话或互联网费用都可以申报为税收减免。 |
| **会计费用** | 会计费用是可以税前扣除的。 |

## 资本利得税（CGT）

如果房产投资者在持有房产一年内出售投资房产，他们需要为销售利润缴纳资本利得税。如果持有时间超过一年，您可以享受50%的CGT折扣，这意味着您只需在年度纳税申报表中申报一半的资本增值。
